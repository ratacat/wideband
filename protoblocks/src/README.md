# Protoblock Source Area

Each experiment owns one folder:

```text
protoblocks/src/<protoblock-slug>/
```

Put experiment code, scratch files, and outputs in that folder. Keep run artifacts under:

```text
protoblocks/src/<protoblock-slug>/runs/<run-id>/
```

Do not store secrets here. Do not add shared helpers until at least two completed experiments prove the helper belongs in shared code.
